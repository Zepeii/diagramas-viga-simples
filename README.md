# diagramas-viga-simples
Análise de Vigas Simples em Python

Este projeto consiste em um código em Python para análise de vigas simples, fornecendo:

reações de apoio;
diagramas de esforço normal;
diagramas de esforço cortante;
diagramas de momento fletor.

Os resultados são obtidos a partir dos carregamentos e condições de contorno definidos pelo usuário, com suporte a uma interface interativa desenvolvida em Streamlit.


Formas de utilização

O código pode ser utilizado de duas maneiras:

1. Utilização com interface interativa (Streamlit)
Copie ou clone o repositório para sua máquina.
No terminal, navegue até a pasta do projeto.

Execute o comando:
streamlit run main.py

Uma interface gráfica será aberta no navegador, permitindo a modelagem da viga (comprimento, apoios e carregamentos) e a visualização dos resultados gerados pelo código.


2. Utilização direta em Python (sem interface)

O código também pode ser utilizado diretamente em Python, sem o uso da interface do Streamlit.
Nesse caso, é possível executar as funções principais tomando como base os exemplos utilizados durante o desenvolvimento do código, adaptando-os conforme a necessidade do usuário.
